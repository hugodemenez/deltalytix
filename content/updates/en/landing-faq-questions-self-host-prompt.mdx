# Homepage Questions with a self-host agent prompt

The homepage FAQ section is now titled **Questions**. The accordion opens more than one answer at a time, so you can compare details without closing what you already read.

**Is it possible to run Deltalytix locally?** now points visitors toward local / self-hosted dashboard mode. Local mode supports CSV and PDF imports with seeded demo data; live broker sync (Rithmic, Tradovate, Thor) stays on the cloud app.

From that answer, expand **Agent setup prompt**, then use **Copy prompt** or **Open with** to send it to Cursor, ChatGPT, or Claude so an agent can bootstrap the environment for you.
