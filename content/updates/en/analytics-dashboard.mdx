# Analytics Dashboard Launch

We're excited to announce the launch of our new analytics dashboard, providing real-time insights into your data.

## Key Features

- Real-time data visualization
- Customizable widgets
- Export capabilities
- Dark mode support

> The dashboard provides instant insights into your business metrics, helping you make data-driven decisions faster than ever before.

### Implementation Details

Here's a quick example of how to use the dashboard API:

```typescript
import { DashboardClient } from '@/lib/dashboard'

// Initialize the dashboard client
const dashboard = new DashboardClient({
  apiKey: process.env.DASHBOARD_API_KEY,
})

// Fetch real-time metrics
const metrics = await dashboard.getMetrics({
  startDate: new Date('2024-01-01'),
  endDate: new Date(),
  interval: '1h'
})
```
