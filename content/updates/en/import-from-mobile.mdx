# Import your trades from your phone

![Import dialog on a phone-sized screen](/updates/pr-249/en/import-from-mobile.png)

The **Import** button is now available in the mobile navbar, so you can add trades or start a sync without switching to a desktop.

The platform picker and import dialog were reworked to fit narrow screens: the list of platforms, the tutorials, and the footer buttons adapt to the width instead of getting cut off. Picking a platform and following its setup steps works the same on a phone as on desktop.
