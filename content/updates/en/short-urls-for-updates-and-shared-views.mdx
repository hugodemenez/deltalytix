# Bookmark-friendly shortcuts for updates and shared views

You can now start from the shorter paths **`/updates`**, **`/updates/<slug>`**, or **`/shared/<slug>`** without hand-typing **`en`** first. The platform issues a quick redirect into the English locale route so the page loads exactly like the long-form URL you’re used to.

That’s handy when you’re sharing with teammates who only speak English, copy links from older docs, or want friendlier URLs in slide decks.

Once you’re on the page you can still jump to another language from the regular language switcher if you need it.
