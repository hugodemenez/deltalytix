# Shared performance links ship a rich preview card

When you drop one of those **published performance links** anywhere that builds a snippet (think Slack threads, Discord, text messages, whatever), preview bots grab the **right title**, **short description**, **canonical URL**, and a **purpose-built preview illustration** tuned for shared dashboards instead of falling back to a vague site-wide default.

Older marketing domains that typo’d “Deltalytix” now normalize back to the live app host, so people who saved legacy bookmarks still end up with honest cards that point at the product you expect.

If you’ve been hesitating to share dashboards because the preview looked broken, this pass is meant to make the outside world match what guests already see after they click through.
