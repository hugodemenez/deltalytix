# Live Rithmic balances in the accounts table

![Accounts table with Rithmic balance column](/updates/pr-288/en/rithmic-live-balance-display.png)

If you connect Rithmic accounts, the **Accounts** table now shows a dedicated **Rithmic balance** column with live values pulled from your broker. Tap refresh on any row to fetch the latest balance without leaving the dashboard.

This sits next to your tracked balance and payout columns, so you can compare what Rithmic reports right now against what Deltalytix has recorded from synced trades.
