# Enhanced Trade Table & Analytics Features

We're excited to announce a major upgrade to the trade table and analytics experience, bringing powerful editing capabilities and improved visualization tools.

## Trade Table Enhancements

### Bulk Edit Panel
Efficiently modify multiple trades at once with our new bulk editing system:

**Time Offset Operations:**
- Adjust entry and exit times for selected trades
- Add or subtract hours/minutes in batch
- Perfect for timezone corrections
- Preview changes before applying

**Instrument Modifications:**
- Replace instrument names across multiple trades
- Trim unwanted characters (prefixes, suffixes)
- Add prefixes or suffixes to instruments
- Standardize naming conventions

### Inline Editing
Edit trades directly in the table without opening separate modals:

**Editable Fields:**
- Trade times (entry and exit)
- Instrument names
- Click to edit, press Enter to save
- Escape to cancel changes
- Visual feedback during editing

**Improved UX:**
- Hover effects show editable fields
- Selected row highlighting
- Consistent background and border colors
- Better visual coherence

### Enhanced Display
- Increased default page size from 10 to 20 rows for better data visibility
- Improved table header styling
- More intuitive row selection
- Thin scrollbar utility for cleaner appearance
- Adjusted input text sizing for better readability
