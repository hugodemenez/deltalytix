# Import dialog that fits your screen

The **Import** dialog layout was reworked so the platform list and the sync panel share space more sensibly. On large screens they sit in a **30 / 70** split—the list on the left, Tradovate, DxFeed, or other import details on the right.

On smaller screens the list gets a fixed height at the top and the sync panel scrolls below it, so buttons and account tables are not squeezed off the side. This applies to every import type in the dialog, not just DxFeed.
