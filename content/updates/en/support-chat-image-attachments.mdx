# Send images in support chat

![Support assistant chat on the help page](/updates/pr-288/en/support-chat-image-attachments.png)

You can now attach screenshots and images in the support assistant on the help page and in the dashboard chat. Send an image on its own or alongside your message—the attachment shows inline in the conversation so support can see exactly what you're looking at.

This makes it easier to report UI issues or ask about something specific on screen without copying image links or describing layouts in text.
