# Tradovate sync: choose which fees count

Tradovate fills carry several fee buckets (commission, exchange, clearing, NFA, and others). In the **Tradovate sync** connection screen you can now turn each type on or off before syncing.

Default behavior stays close to before (commission only), but if your prop or broker tags extra lines on every fill, you can include them so PnL in Deltalytix matches what you expect.
