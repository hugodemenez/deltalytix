# Faster Get Started with loading feedback

![Landing page hero with Get Started button](/updates/pr-288/en/get-started-navigation-feedback.png)

The **Get Started** button on the landing page now responds immediately when you tap it. A loading state appears right away while the sign-in page loads in the background, so the click never feels like it was ignored.

We also added route-level loading indicators for the authentication and dashboard pages, giving clearer feedback during navigation on slower connections.
