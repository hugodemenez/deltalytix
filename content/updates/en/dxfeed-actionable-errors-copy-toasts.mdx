# DxFeed errors you can act on—and copy

DxFeed sync errors are now written in everyday language with **next steps** attached: check demo vs live credentials, pick the matching prop firm, reconnect after an expired session, or contact Support if your firm is missing from the list.

Success and error **toasts** include a **Copy** button so you can paste the full message into a support ticket or note without retyping. Partial sync warnings and "no closed trades yet" cases get the same treatment, so you know whether to wait, reconnect, or try again later.
