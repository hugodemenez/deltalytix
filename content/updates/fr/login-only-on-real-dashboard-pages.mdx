# La connexion se cache vraiment sur le tableau de bord trader

Jusque-là, la passerelle du site vérifiait grossièrement si l’URL contenait la sous-chaîne **`/dashboard`**.

Des pages publiques, par exemple une note dans **`/updates/...`** dont le chemin contient un segment du type **`…/dashboard-widgets`**, pouvaient se retrouver classées comme “zone privée” et te balancer sur l’écran d’auth sans motif.

Les garde-fous vérifient maintenant uniquement les chemins qui pointent réellement vers **`/dashboard`**, **`/dashboard/…`**, **`/teams/dashboard`** et **`/teams/dashboard/…`**, autrement dit là où vivent tes outils de trading.

Les contenus vitrine qui parlent encore de “dashboard” dans le libellé peuvent ainsi rester lisibles avant toute tentative de connexion.
