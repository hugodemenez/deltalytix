# Les billets Mises à jour affichent le bon nom et un lien partageable propre

Les **pages de changelog** accessibles via **Mises à jour** sur le site vitrine affichent désormais **Deltalytix** partout dans les données structurées : fini les erreurs qui faisaient trainer une graphie barrée sous le nez de Google ou des messageries.

Les **URL canoniques** et les équivalents **FR / EN** passent systématiquement par la base officielle du produit au lieu de suppositions figées dans le code, ce qui évite deux versions du même article flottantes côté moteurs de recherche.

Pour les applis qui construisent une carte à partir du lien, chaque billet cite désormais **l’illustration qui lui correspond**, ce qui rapporte une vignette fidèle au titre plutôt qu’un bloc gris passe-partout comme avant.
