# Raccourcis faciles pour les mises à jour et les vues partagées

Tu peux coller **`/updates`**, **`/updates/<slug>`** ou **`/shared/<slug>`** sans ajouter **`/en/`** tout de suite. Le site redirige rapidement vers l’entrée équivalente en langue anglaise, puis le rendu est identique à ce que donnait déjà la version complète du lien.

C’est pratique pour les anciens documents, pour les gens qui utilisent encore des favoris très courts ou pour garder une URL courte pendant une présentation.

Une fois dessus tu peux repasser en français depuis le commutateur habituel tant que la locale est supportée.
