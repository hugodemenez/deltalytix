# Erreurs DxFeed actionnables—et copiables

Les erreurs de sync DxFeed sont desormais redigees en langage simple avec des **prochaines etapes** : verifier demo vs live, choisir la bonne prop firm, reconnecter apres expiration, ou contacter le Support si votre firme manque dans la liste.

Les **toasts** de succes et d'erreur incluent un bouton **Copier** pour coller le message complet dans un ticket support sans retaper. Les avertissements de sync partielle et les cas "pas encore de trades fermes" suivent la meme logique, pour savoir s'il faut attendre, reconnecter ou reessayer.
