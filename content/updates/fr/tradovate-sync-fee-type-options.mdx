# Sync Tradovate : choisir les frais inclus

Les executions Tradovate comportent plusieurs lignes de frais (commission, bourse, compensation, NFA, etc.). Dans l'ecran **synchronisation Tradovate**, vous pouvez maintenant cocher ou decocher chaque type avant de lancer une sync.

Par defaut, on reste proche de l'ancien comportement (commission seule), mais si votre prop ou courtier ajoute des lignes sur chaque fill, vous pouvez les inclure pour que le PnL dans Deltalytix colle a ce que vous suivez.
