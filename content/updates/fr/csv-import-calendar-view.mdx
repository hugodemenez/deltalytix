# Import de données CSV avec vue calendrier

Deltalytix commence par l'essentiel : importer vos trades et voir vos résultats en un coup d'oeil.

## Fonctionnement

Uploadez un fichier CSV contenant vos trades. Le système valide les données et les affiche sur un calendrier interactif. Chaque jour montre votre profit ou perte, facilitant l'identification de tendances.

## Fonctionnalités principales

- **Import CSV** : Uploadez vos données depuis n'importe quel broker exportant en CSV
- **Validation automatique** : Les erreurs et incohérences sont détectées à l'import
- **Vue calendrier** : Visualisez votre PnL quotidien sur une grille mensuelle
- **Courbe d'équité** : Suivez la croissance de votre compte dans le temps
- **Stockage sécurisé** : Données stockées dans PostgreSQL avec chiffrement

## Pour commencer

1. Exportez vos trades en CSV depuis votre broker
2. Uploadez le fichier dans Deltalytix
3. Mappez les colonnes si nécessaire (ou laissez le système les détecter)
4. Consultez vos trades sur le calendrier et la courbe d'équité
