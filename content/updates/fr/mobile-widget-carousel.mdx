# Widgets mobile : carrousel de cartes plein écran

![Carrousel de widgets mobile sur le tableau de bord](/updates/pr-288/fr/mobile-widget-carousel.png)

Sur mobile, l'onglet **Widgets** ne compresse plus tous les graphiques dans une grille étroite. Chaque widget occupe une carte plein écran que vous parcourez verticalement—graphiques, calendriers et stats sont assez grands pour être lus sans pincer ou plisser les yeux.

Le carrousel respecte l'ordre de votre mise en page et indique le widget actif, pour une navigation sur téléphone plus proche de feuilleter des cartes que de faire défiler une mosaïque miniature.
