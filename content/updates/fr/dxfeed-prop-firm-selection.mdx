# Sync DxFeed : choisir votre prop firm d'abord

La fenetre **Connecter un compte DxFeed** commence desormais par une liste **Prop firm**. Choisissez la firme ou vous tradez—Miltraders et Phoenix Trader Funding sont disponibles aujourd'hui—puis connectez-vous avec le meme email et mot de passe que sur leur plateforme.

Les trades sont recuperes sur le serveur d'historique de cette firme, pas sur un site DxFeed generique. Si vos identifiants appartiennent a une autre firme que celle selectionnee, un message clair vous le signale. Si votre firme n'apparait pas encore, l'ecran indique comment contacter le **Support** pour qu'on l'ajoute.
