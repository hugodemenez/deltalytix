# Le graphique d'accueil relie les commits aux mises à jour livrées

La section **Ouvert et Transparent** de la [page d'accueil](/fr) relie désormais l'activité du dépôt à ce qui a réellement été livré dans le produit.

Survolez une semaine sur desktop — ou ouvrez un mois sur mobile — pour voir le total de commits, les **Lignes ajoutées** et **Lignes supprimées** lorsque les données de fréquence de code GitHub sont disponibles, ainsi que les **Mises à jour produit** qui mènent directement à l'entrée de changelog correspondante. Si une période n'a pas encore de publication liée, **Parcourir le changelog** mène toujours à la timeline complète.

![Détail hebdomadaire du graphique avec liens vers les mises à jour produit](/updates/pr-316/fr/landing-contribution-graph-product-updates.png)

Cette évolution s'appuie sur le graphique hebdomadaire déjà en place : vous pouvez toujours naviguer entre les années et comparer l'activité sur **main** et **beta**, avec en plus un lien direct entre le développement et les annonces utilisateur.

[Découvrir la section open source](/fr#open-source) sur la page d'accueil.
