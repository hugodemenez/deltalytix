Fini les exports CSV manuels. Connectez votre compte Rithmic et vos trades se synchronisent automatiquement chaque heure.

## Fonctionnement

Entrez vos identifiants Rithmic dans les paramètres du compte. Deltalytix se connecte à l'API Rithmic et récupère vos trades automatiquement. Les nouveaux trades apparaissent dans votre dashboard dans l'heure suivant leur exécution.

## Fonctionnalités principales

- **Sync horaire** : Les trades sont récupérés automatiquement chaque heure
- **Import historique** : Récupérez jusqu'à 400 jours de trades passés
- **Multi-comptes** : Connectez plusieurs comptes Rithmic
- **Feedback temps réel** : Voyez le statut et la progression de la sync
- **Connexion sécurisée** : Identifiants chiffrés et stockés de manière sécurisée

## Pour commencer

1. Allez dans les paramètres du compte
2. Sélectionnez "Connecter un compte Rithmic"
3. Entrez vos identifiants Rithmic
4. Confirmez la connexion

Vos trades se synchroniseront automatiquement à partir de ce moment.
