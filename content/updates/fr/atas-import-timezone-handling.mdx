# Import ATAS : heures des trades selon votre fuseau

Les heures d'ouverture et de cloture du journal ATAS sont interpretees dans **le fuseau de votre profil**, puis enregistrees de facon coherente pour les graphiques, filtres et le tableau des trades. Cela couvre les cellules date Excel, les chaines `AAAA-MM-JJ HH:mm` et les formats de date parfois renvoyes par ATAS.

Apres l'import, les horaires doivent correspondre a ce que vous voyez dans ATAS, sans decalage d'une heure ni changement de jour quand votre fuseau n'est pas UTC.
