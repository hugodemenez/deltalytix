# Saisie manuelle des trades avec calcul automatique des commissions

Pour les brokers sans synchronisation automatique ou quand vous devez ajouter des trades non capturés, la saisie manuelle offre une solution complète.

## Fonctionnement

Sélectionnez "Saisie manuelle" dans les options d'import. Remplissez les détails du trade via le formulaire interactif. Le système calcule les commissions automatiquement en se basant sur vos trades précédents pour le même instrument.

## Formulaire de saisie

Entrez tous les détails du trade :

- **Instrument** : Symbole ou nom du contrat
- **Sens** : Long ou short
- **Heure d'entrée** : Quand vous avez ouvert la position
- **Heure de sortie** : Quand vous l'avez fermée
- **Prix d'entrée** : Votre prix d'entrée
- **Prix de sortie** : Votre prix de sortie
- **Quantité** : Nombre de contrats ou actions
- **Commission** : Calculée auto ou saisie manuellement

## Calcul auto des commissions

Le système analyse vos trades historiques pour le même instrument et calcule la commission moyenne par contrat. Cette valeur est suggérée automatiquement mais peut être modifiée.

## Fonctionnalités principales

- **Validation du formulaire** : Les champs requis sont validés avant sauvegarde
- **PnL temps réel** : Voyez le profit/perte calculé pendant la saisie
- **Édition et suppression** : Modifiez ou supprimez les trades saisis à tout moment
- **Statistiques** : Les trades manuels sont inclus dans toutes les analyses
- **Assignation de compte** : Assignez les trades à des comptes spécifiques
