# Le calendrier et le tableau s'accordent sur vos dates

![Widget calendrier avec le fuseau horaire de votre profil](/updates/pr-249/calendar-table-timezone-date-fix-widgets.png)

Certains trades—surtout les positions ouvertes tard le soir—pouvaient apparaître le mauvais jour dans le calendrier et l'agenda alors que le tableau affichait la bonne date. Le calendrier regroupait les jours dans un fuseau horaire différent de celui utilisé par le tableau.

![Dates du tableau des trades alignées avec le calendrier](/updates/pr-249/calendar-table-timezone-date-fix-table.png)

Les deux vues utilisent maintenant le fuseau de votre profil de façon cohérente, donc un trade tombe toujours le même jour partout : le calendrier, la vue hebdomadaire, la fenêtre du jour et le tableau. Les totaux de PnL par jour dans le calendrier correspondent à ce que montre le tableau.
