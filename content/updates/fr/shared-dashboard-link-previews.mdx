# Les liens de performance partagés montrent un vrai aperçu

Quand tu colles une URL d’**analyse partagée** dans Slack, Discord, X, iMessage ou tout réseau qui fabrique une carte à partir du lien, les métadonnées reprennent le **titre** et **la petite description de cette vue** et appellent une **image d’aperçu** calée dessus.

Les URL canoniques et les vignettes renvoyées aux réseaux utilisent désormais le bon domaine Deltalytix, même quand quelqu’un ressort un ancien lien écrit différemment. En pratique : moins de cartes vierges ou de miniatures envoyées depuis un bloc qui ne reflète pas ta page.
