# Table des comptes : colonne frais totaux

La vue **tableau** des comptes inclut une colonne **Frais totaux**. Elle part des frais d'activation et, pour les comptes factures au mois, multiplie le prix mensuel par la duree d'activite estimee (du premier au dernier trade), pour donner un ordre de grandeur du cout total.

Les resumes de groupe additionnent la meme chose sur les comptes du groupe, pratique pour comparer rapidement les depenses prop aux soldes et paiements.
