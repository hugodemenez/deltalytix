# Pages de performance partagées plus fiables

Ouvrir un **lien de performance partagé** redevient fluide, même quand le compte sous-jacent contient un gros historique de trades. Les pages qui échouaient avec un timeout base de données se chargent normalement, et le compteur de vues continue de s'incrémenter en arrière-plan.

Les aperçus de lien fonctionnent aussi quand vous collez une URL partagée dans Slack, iMessage ou les réseaux sociaux. L'image Open Graph des pages partagées ne plante plus, donc vos contacts voient une vraie carte d'aperçu plutôt qu'une vignette vide ou générique.
