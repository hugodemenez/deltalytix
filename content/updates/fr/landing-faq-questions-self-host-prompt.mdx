# Questions sur la page d'accueil et prompt d'auto-hébergement

La section FAQ de la page d'accueil s'intitule désormais **Questions**. L'accordéon peut laisser plusieurs réponses ouvertes à la fois, pour comparer les détails sans refermer ce que vous venez de lire.

**Est-il possible d'exécuter Deltalytix localement ?** oriente désormais les visiteurs vers le mode tableau de bord local / auto-hébergé. En local, les imports CSV et PDF fonctionnent avec des données de démo ; la synchronisation courtier en direct (Rithmic, Tradovate, Thor) reste réservée à l'application cloud.

Depuis cette réponse, développez le **Prompt de configuration pour agent**, puis utilisez **Copier le prompt** ou **Ouvrir avec** pour l'envoyer dans Cursor, ChatGPT ou Claude afin qu'un agent configure l'environnement à votre place.
