# Mises en page partagees : modifier les comptes visibles

Dans **Mises en page partagees** (la ou vous gerez les liens publics vers votre tableau de bord), chaque carte permet d'ouvrir un editeur de **comptes**. Choisissez parmi les numeros de compte deja presents dans vos trades, ajoutez ou retirez-les, puis enregistrez.

Pratique quand vous changez de comptes ou corriger une erreur : la bonne selection est visible pour coach, investisseur ou coequipier, sans refaire un lien de zero.
