# Mode sombre et système de thèmes

Les sessions d'analyse peuvent être longues. Le mode sombre réduit la fatigue oculaire, surtout pour le trading de nuit.

## Fonctionnement

Basculez entre les thèmes clair et sombre depuis la barre de navigation. Votre préférence est sauvegardée et persiste entre les sessions. Le système peut aussi suivre automatiquement les paramètres de votre OS.

## Fonctionnalités principales

- **Thème clair** : Interface lumineuse pour une utilisation de jour
- **Thème sombre** : Luminosité réduite pour les sessions nocturnes
- **Synchronisation système** : Suit automatiquement la préférence de votre OS
- **Sélection persistante** : Votre choix est mémorisé
- **Graphiques optimisés** : Tous les graphiques adaptent leurs couleurs pour rester lisibles

## Adaptations visuelles

- Les couleurs des graphiques s'ajustent pour le contraste
- Le texte reste lisible dans toutes les conditions
- Les tableaux conservent leur clarté
- La hiérarchie visuelle est préservée
