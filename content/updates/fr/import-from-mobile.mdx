# Importez vos trades depuis votre téléphone

![Fenêtre d'import sur un écran de téléphone](/updates/pr-249/fr/import-from-mobile.png)

Le bouton **Import** est désormais disponible dans la barre de navigation mobile, pour ajouter des trades ou lancer une sync sans passer sur un ordinateur.

Le sélecteur de plateformes et la fenêtre d'import ont été revus pour les écrans étroits : la liste des plateformes, les tutoriels et les boutons du bas s'adaptent à la largeur au lieu d'être coupés. Choisir une plateforme et suivre ses étapes de configuration fonctionne pareil sur téléphone et sur ordinateur.
