# Envoyer des images dans le chat support

![Chat de l'assistant support sur la page d'aide](/updates/pr-288/fr/support-chat-image-attachments.png)

Vous pouvez désormais joindre des captures d'écran et des images à l'assistant support sur la page d'aide et dans le chat du tableau de bord. Envoyez une image seule ou avec votre message—la pièce jointe s'affiche dans la conversation pour que le support voie exactement ce que vous voyez.

Cela facilite le signalement de problèmes d'interface ou les questions sur un écran précis, sans copier de liens d'images ni décrire la mise en page par écrit.
