# Ouvrir la boîte mail fonctionne sur iOS

Après avoir demandé un lien magique, appuyer sur **Ouvrir la boîte mail** sur iOS vous amène désormais dans votre boîte de réception pour retrouver l'e-mail. Auparavant, Mail sur iOS pouvait ouvrir un écran de composition, ce qui rendait la connexion déroutante.

Si la boîte de réception ne peut pas s'ouvrir automatiquement, un toast vous rappelle de vérifier vos e-mails manuellement.
