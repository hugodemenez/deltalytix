Interactive Chart Test

Test and interact with the Deltalytix trading performance chart in real-time

How to use

Use the controls below to send messages to the embedded chart. The chart will update in real-time based on your interactions.

Available Actions:

  • Client-Side: Generate trades locally and send full trade data
  • Server-Side: Send count only, let server generate trades
  • Specific Trade: Add a custom trade via dialog form
  • Reset: Reset to original mock data
  • Clear: Clear all trades from the chart
