Interactive Chart Test

Test and interact with the Deltalytix trading performance chart in real-time

How to use

Use the controls below to send messages to the embedded chart. The chart will update in real-time based on your interactions.

Available Actions:

  • ADD_TRADES - Add random trades or specific trade data
  • RESET_TRADES - Reset to original mock data
  • CLEAR_TRADES - Clear all trades from the chart
  • ADD_PHOENIX_ORDERS - Add Phoenix orders to be processed into trades
